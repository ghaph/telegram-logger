# Telegram Logger

Easily log all chat messages to a file

## Usage

1. Visit [https://my.telegram.org/apps](https://my.telegram.org/apps) and login
2. Create an app and name it whatever you want
3. Copy the API ID and the API hash for later
4. Make sure python is installed
5. Run the logger.py file
6. Fill in all the info it asks you (You only have to enter this once)
7. You're done
